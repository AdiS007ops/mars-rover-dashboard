# mars-rover-dashboard

This repository contains the vision processing and dashboard system for the Mars Rover project. It captures live video and processes the feed to monitor terrain and detect obstacles.

Hardware Used
* Raspberry Pi: Acts as the main computing brain for the mission. 
* USB Webcam: Mounted on the rover to capture live video feeds of the environment.

Software & Libraries
* Python 3: The core programming language running on the Raspberry Pi.
* OpenCV: Used for real-time computer vision tasks, image processing, and rendering the visual dashboard overlay.

Setup and Installation

1. Clone the repository to your Raspberry Pi:
   bash git clone [https://github.com/your-username/mars-rover-dashboard.git](https://github.com/your-username/mars-rover-dashboard.git)
   cd mars-rover-dashboard
